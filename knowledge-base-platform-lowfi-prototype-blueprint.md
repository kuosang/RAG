## 1. 低保真原型统一约定

## 1.1 页面结构统一约定
所有后台型页面建议采用统一布局：

```text
┌──────────────── 顶部导航 / 面包屑 / 全局操作 ────────────────┐
│ Logo | 一级导航 | 全局搜索 | 消息 | 帮助 | 用户中心           │
├─────────────── 左侧导航 ───────────────┬────────────────────┤
│ 二级菜单                              │ 页面主体区          │
│ - 菜单1                               │ 标题 / 筛选 / 操作  │
│ - 菜单2                               │ 统计卡 / 表格 / 表单│
│ - 菜单3                               │ 详情区 / 抽屉 / 弹窗│
└───────────────────────────────────────┴────────────────────┘
```

## 1.2 页面状态统一约定
每个页面都需要考虑这些状态：
- 默认态
- 空状态
- 加载态
- 错误态
- 无权限态
- 成功反馈态

## 1.3 角色统一约定
后续页面中会频繁出现这些角色缩写：
- 平台管理员
- 知识管理员 / 运营
- 编辑人员
- 审核人员
- 业务负责人
- AI 产品经理 / Agent 配置人员
- 工作流编排 / 应用开发
- 普通业务用户

# 页面 01：管理后台首页

## 页面目标
作为平台总入口，帮助管理员和运营快速掌握平台运行状态，发现风险项，并进入高频管理动作。

## 布局结构
```text
┌──────────────────────────────────────────────────────────┐
│ 顶部：平台名称 / 全局搜索 / 消息提醒 / 用户菜单            │
├───────────────┬──────────────────────────────────────────┤
│ 左侧导航      │ 主体区                                     │
│ - 总览        │ ① 核心指标卡                               │
│ - 知识管理    │ ② 待办提醒区                               │
│ - 审核发布    │ ③ 平台趋势图                               │
│ - 维护运营    │ ④ 风险预警区                               │
│ - 检索调试    │ ⑤ 快捷入口区                               │
│ - Agent集成   │ ⑥ 最近活动 / 系统公告                      │
│ - 平台配置    │                                            │
│ - 权限安全    │                                            │
└───────────────┴──────────────────────────────────────────┘
```

## 区块组成
1. 核心指标卡
   - 知识总量
   - 已发布知识数
   - 本周新增数
   - 检索命中率
   - 零结果率
   - Agent 调用量
2. 待办提醒
   - 待审核知识
   - 即将过期知识
   - 维护任务数
   - 同步失败任务
3. 平台趋势图
   - 近 7/30 天知识增长趋势
   - 检索趋势
   - Agent 调用趋势
4. 风险预警
   - 高负反馈知识
   - 低命中空间
   - 权限异常告警
5. 快捷入口
   - 新建知识空间
   - 新建知识
   - 进入审核中心
   - 进入检索调试台
   - 进入 Agent 绑定
6. 最近活动
   - 最近发布
   - 最近审核
   - 最近配置变更

## 关键操作
- 切换时间范围
- 点击指标卡进入明细列表
- 点击待办进入对应处理页
- 一键进入常用管理功能
- 查看告警详情

## 页面跳转关系
- 待审核数 → 审核中心
- 即将过期 → 维护中心
- 检索命中率 → 质量分析 / 检索分析
- Agent 调用量 → 调用监控页
- 新建知识空间 → 知识空间管理页

## 适配角色与状态
### 适配角色
- 平台管理员
- 知识管理员 / 运营
- 安全审计人员（部分数据）

### 页面状态
- 默认态：展示完整平台看板
- 空状态：提示“当前暂无统计数据，请先创建知识空间或导入知识”
- 无权限态：隐藏敏感指标，仅展示可见板块

# 页面 02：知识空间管理页

## 页面目标
统一管理知识空间、目录结构、空间负责人和空间权限，是整个平台知识组织的总入口。

## 布局结构
```text
┌──────────────── 页面标题：知识空间管理 ────────────────────┐
│ 操作栏：新建空间 | 导入空间配置 | 批量操作 | 筛选            │
├──────────────────────────────────────────────────────────┤
│ 左：空间树 / 业务域树   │ 右：空间列表 / 空间详情            │
│ - 产品知识库            │ ① 列表表格                         │
│ - 客服知识库            │ ② 负责人/权限摘要                  │
│ - 运营知识库            │ ③ 最近更新时间                      │
│                         │ ④ 操作按钮：编辑/权限/归档         │
└──────────────────────────────────────────────────────────┘
```

## 区块组成
1. 顶部筛选区
   - 业务域
   - 空间状态
   - 负责人
   - 更新时间
2. 左侧空间树
   - 支持按业务域、部门、产品线分层
3. 空间列表区
   - 空间名称
   - 空间编码
   - 业务域
   - 知识数量
   - 负责人
   - 权限类型（公开/受限）
   - 最近更新时间
4. 详情侧边栏 / 抽屉
   - 空间说明
   - 目录结构
   - 已绑定 Agent 数
   - 权限规则摘要

## 关键操作
- 新建空间
- 编辑空间基础信息
- 配置目录结构
- 配置空间权限
- 指定负责人
- 查看已绑定 Agent
- 空间归档 / 启停用

## 页面跳转关系
- 新建空间 → 空间新建弹窗 / 独立页
- 点击空间名称 → 空间详情页 / 抽屉
- 权限配置 → 权限配置页
- 查看绑定 Agent → Agent 绑定页
- 查看空间知识 → 我的知识列表页

## 适配角色与状态
### 适配角色
- 平台管理员
- 知识管理员 / 运营

### 页面状态
- 默认态：列表 + 树结构并存
- 空状态：展示“暂无知识空间，立即创建第一个空间”
- 错误态：空间树加载失败，支持重试
- 无权限态：仅可查看授权空间

# 页面 03：新建知识页

## 页面目标
让编辑人员快速、规范地创建知识内容，并在录入时尽量一次性补全结构化字段。

## 布局结构
```text
┌──────────────── 页面标题：新建知识 ───────────────────────┐
│ 顶部：选择空间 | 选择模板 | 保存草稿 | 提交审核             │
├──────────────────────────────────────────────────────────┤
│ 左：正文编辑区                │ 右：结构字段区              │
│ 标题输入                      │ - 分类                       │
│ 摘要输入                      │ - 标签                       │
│ 富文本正文                    │ - 责任人                     │
│ 附件上传                      │ - 生效/失效时间              │
│                               │ - 适用对象/可见范围          │
│                               │ - 来源信息                   │
└──────────────────────────────────────────────────────────┘
```

## 区块组成
1. 模板选择区
   - FAQ 模板
   - 产品说明模板
   - SOP 模板
   - 规则模板
   - 案例模板
2. 内容编辑区
   - 标题
   - 摘要
   - 正文
   - 图片 / 附件
3. 结构字段区
   - 知识空间
   - 分类 / 标签
   - 责任人
   - 适用角色
   - 优先级
   - 生效时间 / 失效时间
   - 来源类型 / 来源链接
4. 校验提示区
   - 缺字段提醒
   - 风险提示
   - 重复知识提醒（可预留）

## 关键操作
- 选择模板创建
- 自动保存草稿
- 手动保存草稿
- 上传附件
- 提交审核
- 预览知识详情
- 退出前未保存提醒

## 页面跳转关系
- 保存草稿 → 我的知识 / 草稿箱
- 提交审核 → 审核中心待审核列表
- 预览 → 知识详情预览页
- 取消创建 → 返回工作台首页 / 我的知识

## 适配角色与状态
### 适配角色
- 编辑人员
- 知识管理员 / 运营

### 页面状态
- 默认态：模板已选，展示对应字段
- 空状态：未选择模板，先提示选模板
- 草稿恢复态：提示“检测到未完成草稿，是否继续编辑”
- 校验失败态：高亮缺失字段和错误项

# 页面 04：知识详情编辑页

## 页面目标
用于已创建知识的编辑、版本更新、结构化补充和历史追踪，是知识生命周期中的核心操作页。

## 布局结构
```text
┌──────────────── 页面标题：知识详情编辑 ───────────────────┐
│ 顶部：状态标签 | 保存 | 提交审核 | 发布新版本 | 更多操作     │
├──────────────────────────────────────────────────────────┤
│ 左：主内容区                  │ 右：辅助信息区              │
│ ① 标题/摘要/正文              │ ① 基础属性卡                │
│ ② 内容分段/片段预览           │ ② 发布信息                  │
│ ③ 附件区                      │ ③ 版本历史                  │
│                               │ ④ 维护记录                  │
│                               │ ⑤ 反馈摘要                  │
└──────────────────────────────────────────────────────────┘
```

## 区块组成
1. 顶部状态条
   - 草稿 / 审核中 / 已发布 / 已下线
   - 当前版本号
   - 最近更新时间
2. 主内容编辑区
   - 标题、摘要、正文
   - 片段预览
   - 关联知识
3. 侧边信息区
   - 责任人
   - 审核人
   - 发布时间
   - 生效/失效时间
   - 版本列表
   - 反馈统计
4. 底部操作区
   - 保存修改
   - 提交审核
   - 生成新版本
   - 下线
   - 删除（权限控制）

## 关键操作
- 编辑正文与字段
- 查看版本差异
- 提交新版本审核
- 查看用户反馈
- 查看引用情况
- 执行下线 / 归档

## 页面跳转关系
- 版本历史 → 版本对比页
- 反馈摘要 → 反馈处理页
- 查看引用情况 → 引用链路页 / 调用监控页
- 提交审核 → 审核详情页

## 适配角色与状态
### 适配角色
- 编辑人员
- 知识管理员 / 运营
- 审核人员（只读或带审核入口）

### 页面状态
- 草稿态：可完全编辑
- 审核中：默认锁定关键字段，支持撤回
- 已发布：支持“发起新版本”
- 已下线：只读 + 支持恢复

# 页面 05：审核详情页

## 页面目标
让审核人员高效判断知识是否可发布，并清楚看到本次提交相较上一版本的变更点。

## 布局结构
```text
┌──────────────── 页面标题：审核详情 ───────────────────────┐
│ 顶部：审核状态 | 提交人 | 提交时间 | 通过/退回/转交         │
├──────────────────────────────────────────────────────────┤
│ 左：知识内容预览              │ 右：审核操作区              │
│ ① 当前版本内容                │ ① 审核意见输入框            │
│ ② 差异高亮                    │ ② 风险提示                  │
│ ③ 来源与字段信息              │ ③ 审核记录                  │
└──────────────────────────────────────────────────────────┘
```

## 区块组成
1. 提交摘要区
   - 提交人
   - 提交版本
   - 提交时间
   - 所属空间
2. 内容对比区
   - 当前内容
   - 上一版本内容（差异高亮）
   - 结构字段差异
3. 审核操作区
   - 审核意见
   - 通过
   - 退回修改
   - 转交他人
4. 风险提示区
   - 缺失字段
   - 生效时间异常
   - 过期配置问题
   - 敏感词提醒（预留）

## 关键操作
- 对比版本差异
- 输入审核意见
- 通过审核
- 退回修改
- 转交审核
- 查看历史审核记录

## 页面跳转关系
- 点击知识标题 → 知识详情只读页
- 查看历史版本 → 版本对比页
- 审核通过 → 返回审核列表，并进入发布态流程
- 退回修改 → 回到知识详情编辑页

## 适配角色与状态
### 适配角色
- 审核人员
- 业务负责人（关键知识审批）
- 知识管理员（查看）

### 页面状态
- 待审核态：可操作
- 已通过态：只读 + 显示操作记录
- 已退回态：只读 + 显示退回原因
- 无权限态：仅可查看被授权任务

# 页面 06：搜索结果页

## 页面目标
帮助普通业务用户快速找到目标知识，同时让结果“可理解、可筛选、可反馈”。

## 布局结构
```text
┌──────────────── 搜索框 / 热词 / 搜索建议 ──────────────────┐
├────────── 左侧筛选 ──────────┬─────────────────────────────┤
│ 空间/分类/标签/时间/类型     │ 结果列表区                  │
│                              │ ① 命中总数                  │
│                              │ ② 结果卡片                  │
│                              │ ③ 引用片段高亮              │
│                              │ ④ 推荐搜索 / 兜底建议       │
└─────────────────────────────┴─────────────────────────────┘
```

## 区块组成
1. 搜索区
   - 关键词输入
   - 联想词
   - 搜索历史
2. 左侧筛选区
   - 知识空间
   - 分类
   - 知识类型
   - 标签
   - 更新时间
3. 结果卡片区
   - 标题
   - 摘要
   - 高亮命中片段
   - 来源空间
   - 更新时间
   - 知识类型
4. 兜底区
   - 没找到？推荐相关问题
   - 提交反馈 / 求助

## 关键操作
- 搜索关键词
- 切换筛选条件
- 进入知识详情
- 收藏知识
- 反馈结果无帮助
- 改写搜索词（由系统推荐）

## 页面跳转关系
- 点击结果卡片 → 知识详情页
- 点击推荐问题 → 重新发起搜索
- 点击来源空间 → 分类导航页 / 空间页

## 适配角色与状态
### 适配角色
- 普通业务用户
- 客服 / 销售 / 运营
- 内部员工

### 页面状态
- 默认态：显示相关结果
- 零结果态：显示推荐问题和反馈入口
- 加载态：骨架屏
- 无权限结果态：只展示有权查看内容

```text
│ ③ 相关知识推荐                   │ ③ 版本号                  │
│                                   │ ④ 适用范围                │
│                                   │ ⑤ 反馈入口                │
└──────────────────────────────────────────────────────────┘
```

## 区块组成
1. 顶部信息
   - 标题
   - 所属分类
   - 面包屑
   - 收藏 / 分享

   - 摘要
   - 正文内容
   - 附件下载
   - 相关知识推荐
3. 侧边信息区
   - 来源
   - 责任人（可视情况隐藏）
   - 发布时间 / 更新时间
   - 当前版本
   - 适用对象
4. 底部反馈区
   - 是否解决问题
   - 点赞 / 点踩
   - 提交纠错

## 关键操作
- 查看正文
- 下载附件
- 收藏
- 反馈有效 / 无效
- 查看相关知识
- 查看版本信息

## 页面跳转关系
- 相关知识 → 其他知识详情页
- 反馈纠错 → 反馈提交页 / 弹窗
- 来源链接 → 原始来源跳转（若配置）

## 适配角色与状态
### 适配角色
- 普通业务用户
- 客服 / 销售 / 运营
- 内部员工
- Agent 回答结果引用查看者

### 页面状态
- 默认态：完整展示正文
- 精简态：当从 Agent 引用弹层进入时，优先展示命中片段
- 过期态：展示“该知识已过期/已下线”的提示

# 页面 08：Query 测试页

## 页面目标
让 AI 产品经理和检索配置人员以“实验台”方式验证同一个问题在不同知识空间、不同策略下的召回表现。

## 布局结构
